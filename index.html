HTML

  • Think about when developing a webpage is building a sturdy "frame" using HTML, then, using CSS, start thinking about the colors and styles you might want to add to make it uniquely your own.Finaly, add JS to finally enable you and the user to interact with it
  • HTML attributes are always defined in the beginning element, never the closing element
  • The following element types are commonly used to add content to pages (Text elements, Image elements, Header elements, List elements)
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Elements that are nested inside other elements are called child elements
  • The element that holds all the children known as the parent element
  • An element refers to the entire element from opening tag to closing tag
  • A tag refers only to what appears inside the angle brackets
  • When using headings, it is important that we use heading tags in order from h1 to h6 to make the page accessible to users who access the page using screen readers or other devices
  • The biggest heading, h1, should only appear once on each page
  • The alt text is important for screen readers and other assistive technologies. Alt text can also improve search engine rankings for your website
  • To comment your code in VSC in Mac, select Command + K + C
  • to Un-comment code, select Command + K + U

CSS

  • What I have learned so far?
  • Cascading Style Sheets (CSS), which is a programming language used by developers to define how webpage content is presented to users
  • CSS defines the colors, aesthetics, visual layout, and organization
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet
  • Inline styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only
      • example :? h1 style="color:red;" Prework Study Guide
  • internal style sheet contains CSS rules for the webpage in the head section of the HTML file. The rules only apply to that page, and you can configure CSS classes and IDs, which are used to identify elements, to style multiple elements in the page code
  • The most common way to style a webpage is by using an external CSS style sheet
      • It allows developers to keep all their CSS rules in a separate file, which makes design changes easier
      When using an external style sheet, we link the file using a link element placed in the HTML's head,
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector
      • Then the class attribute can be assigned to the element in the HTML, enabling us to freely choose which elements will be assigned the CSS rule
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

  • What I have learned so far?
  • Git is a distributed version control system that tracks changes in any set of computer files
  • usually used for coordinating work among programmers who are collaboratively developing source code during software development
  • Its goals include speed, data integrity, and support for distributed, non-linear workflows
  • An SSH key is an access credential for the SSH (secure shell) network protocol
  • This authenticated and encrypted secure network protocol is used for remote communication between machines on an unsecured open network
  • SSH is used for remote file transfer, network management, and remote operating system access

  • Learned how to create an SSH Key. For more info, please visit Github online notes

  • Follow these steps to generate an SSH key on MAC:
      ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
      You will then be prompted to "Enter a file in which to save the key."
      You can specify a file location or press “Enter” to accept the default file location.
      Enter passphrase (empty for no passphrase): [Type a passphrase]
      Enter same passphrase again: [Type passphrase again]

    To add the new SSH key to the ssh-agent,do the following:

  • First ensure the ssh-agent is running by executing:
      • $ eval "$(ssh-agent -s)"
      Once the ssh-agent is running the following command will add the new SSH key to the local SSH agent:
      ssh-add -K /Users/you/.ssh/id_rsa
      Now, the new SSH key is registered and ready to use!

    To add your SSH private key to the ssh-agent and store your passphrase in the keychain, type

      ssh-add --apple-use-keychain ~/.ssh/id_ed25519
